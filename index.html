Google Sheets
La app lee y escribe en tu Google Sheet usando una Netlify Function como proxy. La API Key de Google no se guarda en el navegador: configúrala solo en Netlify → Site settings → Environment variables como GOOGLE_SHEETS_API_KEY.
Solo se exige si defines APP_PASSCODE en Netlify (o en tu .env local).
Desarrollo local: usa netlify dev para que /.netlify/functions/sheets exista en el mismo origen.
